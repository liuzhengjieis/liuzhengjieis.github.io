邓小平理论的形成

邓小平理论是以邓小平为主要代表的中国共产党人立足中国又面向世界，总结历史又正视现实、放眼未来，把马克思主义基本原理同中国的国情和时代特征结合起来，在研究新情况、解决新问题的过程中形成和发展起来的。邓小平是全党全军全国各族人民公认的有崇高委婉的卓越领导人，是中国社会主义改革开放和现代化建设的总设计师。

1. 形成条件

  1. 和平与发展成为时代出题（历史背景）

    20世纪70年代的资本主义经济危机，美苏两极格局的对抗冷战，世界各国人民对和平的渴望更加强烈，求和平谋发展成为世界各国人民普遍愿望。东西问题是和平问题，南北问题是发展问题，经济问题（核心问题）。

    当今世界发展迅速，应该以新观点，新思想发展马克思主义。邓小平指出当今世界是开放的世界，中国的发展离不开世界这个时代的趋势。为我们党一心一意搞社会主义现代化建设，实行对外开放以借鉴和吸收人类社会创造的一切文明成果，奠定了科学的基础，也为我们党在复杂变幻的国际局势中冷静沉着，抓住机遇，发展自己，提供了明确的指针。

  2. 社会主义建设的经验教训（历史依据）

    • 第一个五年计划察觉到了苏联模式的弊端。以苏联为鉴，总结自己的经验，探索自己的建设社会主义的道路。这个时期探索中形成了一些正确的理论观点，方针政策和实践经验。
    • 但在这一时期也犯了很多错误，走了不少弯路。大饥荒（经济上急于求成），文化大革命（坚持以阶级斗争为纲）。一，偏离了党的实事求是路线，对国际形势，对社会主义时期主要矛盾和中国现实具体国情作出错误判断。二，对什么是社会主义和如何建设社会主义的问题没有完全搞清楚，因而不可能集中精力发展生产力，也不可能对社会主义的某些制度和体制进行有效改革。
    • 十一届三中全会以后，以邓小平为代表，领导全党和全国人民果断地纠正错误，深刻地分析了它出现的原因，同时又坚决地维护和继承了过去在理论上和时间上所取得的一切积极成果。

  3. 改革开放和现代化建设的实践（现实依据）

    我国改革开放和社会主义现代化建设的崭新实践（前人没有经历过的改革），是人民群众生机勃勃的伟大创造，是理论发展的源泉。邓小平领导党从总结群众成功实践中，从总结某些工作的失误中，把经验上升为理论，揭示了我国社会主义现代化建设的规律，从而创立邓小平理论。

2. 邓小平理论的形成过程

  1. 停止以 “阶级斗争” 为纲要

    1978年12月党的十一届三中全会，重新确立解放思想，实事求是的思想路线。停止使用阶级斗争为纲。

  2. “中国特色社会主义理论” 成为党的理论和实践主题

    1982年邓小平在十二大开幕词中明确：把马克思普遍真理同我国的具体实际联系起来，走自己的到了路，建设有中国特色的社会主义。

  3. 提出公有制基础上的计划经济

    十二大到十三大，随着改革开放和现代化建设实践的全面开展，围绕“什么是社会主义，怎样建设社会主义”的基本理论问题和时间问题进行探讨。

    1984年党的十二届三中全会做出《关于经济体制改革的决定》，提出了社会主义经济是公有制基础上的有计划的商品经济。此后，农村改革到城市改革，由经济体制改革到各方面的体制改革，形成了一系列方针政策以及理论观点。

  4. 轮廓的形成

    1987年召开的党的十三大，第一次系统地论述了我国社会主义初级阶段理论，明确概括和全面阐发了党的“一个中心，两个基本点”的基本路线。从马克思主义的角度对中国特色社会主义理论的主要内容做出了系统概括。

  5. 走向成熟

    1992年南方谈话，深刻总结了十多年改革开放的经验教训，提出了一些列重要论断：

    1. 社会主义本质是解放生产力，发展生产力，消除剥削，消除两极分化，最终共同富裕

    2. “三个有利于”标准

    3. 社会主义可以搞市场经济

    4. 革命是解放生产力，改革也是

    5. 不会坚持社会主义，不改革开放，不发展经济，不改善民生，没有出路

    6. 改革开放要大胆，敢于试验，看准就干

    7. 提倡科学，靠科学才有希望

    8. 两手都要硬，一手抓改革开放，一手抓打击犯罪

    南方谈话是邓小平理论的集大成之作，深刻地回答了当时困扰和束缚人们思想的一些列大问题，推动改革开放和社会主义现代化建设进入新阶段，理论逐步走向成熟

  6. 写入党章，载入宪法

    1992年党的十四大系统总结了改革开放以来十四年取得的巨大成就。

    十四大报告从5九个方面概括了中国特色社会主义理论的主要内容Ⅱ并强调，这个理论第一次比较系统地初步回答了中国这样的经济文化比较落后的国家如何建设社会主义、如何巩固和发展社会主义的一些列基本问题。Ⅲ被认为是马克思列宁主义基本原理于当代中国实际和时代特征相结合的产物，是毛泽东思想的继承和发展。

    1997年召开的党的十五大正式提出“邓小平理论”这一概念。写入党章

    1999年正式载入宪法

邓小平理论的主要内容

1. 邓小平回答的基本问题

  1. 什么是社会主义，怎样建设社会主义

    搞清楚什么是社会主义、怎样建设社会主义，关键是要在坚持社会主义基本制度的基础上进一步认清社会主义的本质。

    邓小平根据马克思主义基本原理和社会主义的实践经验，对这个问题进行了不懈的探索，作出了科学的回答。

  2. 阐述了社会主义的发展目标

    1. 社会主义总目标：解放生产力，发展生产力，消灭剥削，消除两极分化，最终达到共同富裕。社会主义的发展目标以及实现这个目标必须以解放和发展生产力为基础。

      四项基本原则——立国之本，是我国一切进步和发展的基础

      • 坚持社会主义制度
      • 坚持公有制为基础，实行按劳分配原则的社会主义基本经济制度
      • 坚持共产党领导、实行人民民主专政的社会主义基本政治制度
      • 坚持以马克思列宁主义、毛泽东思想为指导的社会意识形态

    2. 经济制度的总目标

      坚持公有制又完善公有制

      **两项根本原则：**Ⅰ社会主义公有制为经济为主体；Ⅱ共同富裕

      在改革中，公有制的实现形式和以公有制为主体的所有制结构，归根到底只
      根据生产力解放和发展的实际要求，根据逐步实现共同富裕的实际进程来确定。

意义：

邓小平关于社会主义本质的概括，遵循了科学社会主义的基本原理，反映了人民的利益和时代的要求，廓清了不合乎时代进步和社会发展规律的模糊观念，摆脱了长期以来拘泥于具体模式而忽略社会主义发展的错误倾向，深化了对科学社会主义的认识。

这对于我们在坚持社会主义基本制度的基础上推进改革，指导改革沿着合乎社会主义本质要求的方向发展，对于建设中国特色社会主义，具有重大的政治意义，理论意义和实践意义。

2. 主要内容

解放思想、实事求是思想路线

1978年党的十一届三中全会召开前夕，邓小平发表《解放思想，实事求是，团结一致向前看》 “如果一切从本本主义出发，思想僵化，迷信盛行，那它就不能前进，它的生机就会停止，就要亡党亡国。”

把党和国家工作重心转移到社会主义现代化建设上来的战略决策，标志着党重新确立了马克思主义的思想路线。1992年邓小平《在南昌、深圳、珠海、上海等地的谈话要点》，是全面改革进程中思想解放的科学总结。

解放思想、实事求是思想路线，有力地推动和保证了改革开放的进行，体现了辩证唯物主义和历史唯物主义的世界观方法论，体现了革命胆略和科学精神的统一，是邓小平理论的活的灵魂，是邓小平理论的精髓。

社会主义和初级阶段理论

实事求是，从实际出发建设社会主义，最大的“实际”便是我国的基本国情：社会主义初级阶段。

党的十三大阐述了中国社会主义处在一个初级阶段，还不发达的阶段，必须一切都要从这个实际出发，根据这个实际制定规划。并系统的论述了社会主义初级阶段理论。

  1. 社会主义初级阶段，就是指我国在生产力落后，商品经济不发达条件下建设社会主义必然要经历的特定阶段，即从我国进入社会主义到基本实现社会主义现代化的整个历史阶段。
  2. 资本主义在中国行不通，走上社会主义道路是历史必然。

社会主义阶段理论基于对中国国情的准确把握，使我们对社会主义建设的长期性，复杂性，艰巨性有了更加清醒的认识，揭示了当代中国的历史方位，是建设中国特色社会主义的总依据，是对马克思主义关于社会主义发展阶段理论的重大发展和重大突破。

党的建设理论

党的十三大报告提出了党在社会主义初级阶段的基本路线：

  1. 领导各族人民
  2. 坚持一个中心，两个基本点
  3. 建成富强民主文明的社会主义现代化国家
  4. 自力更生、艰苦创业

社会主义根本任务的理论

  1. 发展生产力是社会主义根本任务
  2. 中国解决所有问题的关键就是发展生产力
  3. 发展要抓住机遇
  4. 发展离不开科学

“三步走” 战略

“三步走” 是1987年4月邓小平提出的实现现代化的战略。同年10月“三步走”战略构想确定下来。

改革开放理论

改革是第二次革命，使社会主义制度的自我完善和发展。不是推翻阶级，也不是原有经济体制的细枝末节的修补，而是对体制根本性的变革。它的实质和目标，是要从根本上改变束缚我国生产力发展的经济体制，建立充满生机和活力的社会主义新经济体制。

社会矛盾推动了社会主义社会的发展，改革就是社会主义社会发展的直接动力，所以通过改革改变束缚生产力发展的经济体制，促进生产力发展，解决社会主义社会发展动力问题。

开放也是改革，对外开放是建设中国特色社会主义得到一项基本国策。

  • 要认识到对外开放的重要性，“对外开放具有重要意义，任何一个国家要发展，孤立起来，闭关自守是不可能的，不加强国际交往，不引进发达国家的先进经验、先进科学技术和资金，是不可能的。”
  • 对外开放包括对发达国家和发展中国家，包括经济、教育、文化领域。
  • 要正确对待资本主义社会的现代文明，大胆借鉴吸取。
  • 要坚决维护国家主权，自己的事情自己处理，独立自主，自力更生。

社会主义市场经济理论（处理计划经济与市场经济的矛盾）

如何正确处理计划经济与市场经济的矛盾？成为了深化社会主义经济体制改革的主要问题。

改革开放以来，市场经济作用发挥比较充分的地方，经济活力就比较强，发展势态就比较好；正是这是多年来以市场经济为取向的改革，为我们取得建立社会主义市场经济新体制的共识提供了实践基础。（体现出市场经济的好处）

党的十二届三中全会《中共中央关于经济体制改革的决定》提出了社会主义经济是 “公有制基础上的计划经济的商品经济” 的论断。党的十四大根据社会主义也可以搞市场经济的思想，建立了社会主义市场经济体制的改革目标。

社会主义市场经济理论分为几大要点：

  1. 计划经济和市场经济不是划分社会制度的标志。
  2. 计划经济和市场经济都是经济手段，对调节经济活动各有长处短处。
  3. 市场经济作为一种资源配置方式，本身不具有制度属性，可以和不同的社会制度结合，表现出不同的性质。

“两手抓，两手都要硬”

我们的国家已经进入了社会主义现代化建设的新时代，我们要在大幅度提高社会生产力的同时，改革和完善社会主义的经济制度和政治制度，发展高度的社会主义民主和社会主义法制。我们要在发展高度的物质文明的同时，提高全民族的科学文化水平，发展高尚的丰富多彩的文化生活，建设高度的社会主义精神文明。

  1. 党的十二届六中全会初步提出社会主义现代化建设的总体布局：以经济建设为中心，坚定不移建设经济体制改革，政治体制改革，精神文明建设。
  2. 一手抓文明，一手抓精神文明
  3. 一手抓建设，一手抓法制

一国两制

统一是中国历史发展的主流。反对分裂，坚持统一，是中华民族自古以来就有的传统。中国共产党人始终把国家的统一作为自己奋斗的一个重要目标。要想完成祖国的统一，只有实行「一国两制」。

“和平统一、一国两制” 构想的基本内容：

坚持一个中国，这是 “和平统一、一国两制” 的核心，是发展两岸关系和实现和平统一的基础；两制并存，在祖国统一的前提下，国家的主体部分实行社会主义制度，同时在台湾、香港、澳门保持原有的社会制度和生活方式长期不变；高度自治，祖国完全统一后，台湾、香港、澳门作为特别行政区，享有不同于中国其他省、市、自治区的高度自治权，台湾、香港、澳门的同胞各种合法权益将得到切事尊重和维护；尽最大努力争取和平统一，但不承诺放弃使用武力；解决台湾问题，实现祖国完全统一，寄希望于台湾人民。

  1. 一国两制是从中国的实际出发，解决台湾问题、香港问题和澳门问题，实现祖国统一的伟大构想。
  2. 一国两制的提出是从解决台湾问题开始的。
  3. 一国两制首先运用于香港问题
  4. 一国两制的构想是邓小平运用马克思辩证唯物主义和历史唯物主义坚持实事求是，把和平共处的原则用之于解决一个国家的统一问题。

中国问题的关键在于党

建设中国特色社会主义，关键在于坚持、加强和改善党的领导。

  1. 党背负着历史重任，经受着历史的考验，必须发言优良传统，加强自身建设
  2. 执政党的作风，是关系党的生死存亡的重大问题
  3. 党的领导制度、组织制度问题更带有根本性、全局性、稳定性和长期性

邓小平的历史地位

马克思列宁主义、毛泽东思想的继承与发展

  • 继承前人，又突破陈规

    解放思想，实事求是邓小平理论创新思维和理论的核心，是马克思主义基本原理与当代中国实际和时代特征相结合的产物，是马克思列宁主义、毛泽东思想的继承和发展，是全党全国人民集体智慧的结晶。

    邓小平坚持解放思想，实事求是，在新的实践基础上继承前人，突破陈规，开拓了马克思主义的新境界。邓小平坚持辩证唯物主义，和历史唯物主义的立场、观点、方法（马克思主义基本），围绕什么是社会主义，怎样建设社会主义的问题，系统地回答了如何在中国这样一个经济文化比较落后的东方大国建设，巩固和发展社会主义的一系列问题。

中国特色社会会主义的开篇之作

邓小平理论是中国特色社会主义的开篇之作，其理论贯穿了中国社会主义的发展道路、发展阶段、发展动力、根本任务、外部条件，政治保证、战略步骤，党的领导、依靠力量以及祖国统一问题（邓小平理论的十大内容）。科学回答了建设中国特色社会主义的一系列问题。

改革开放和社会主义现代化的科学指南

邓小平理论是邓小平给我们的重要的思想遗产。邓小平理论经过了改革开放和现代化建设实践的检验，被证明是指导中国人民建设中国特色社会主义、保证中国在改革开放中实现国家繁荣富强和人民共同富裕的系统的科学理论。邓小平理论是中国共产党和中国人民的精神财富，是改革开放和社会主义现代化建设的科学指南，是党和国家必须长期坚持的指导思想。
