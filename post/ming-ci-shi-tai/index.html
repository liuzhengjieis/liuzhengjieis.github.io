【动词】时态

2019年12月23日 / 3 min read

对“时态”的概述

时态可以看作是表示动作时间和状态的一种结构，由固定的助动词或分词构成。每个句子都会存在时态的表达，因为时态是英语最为常用的句子结构。

  1. 时态可分为“时”和“态”，即时间和状态；时间为现在时、过去时、将来时。状态为一般态、进行态、完成态。
  2. 一般的时态都是时间和状态构成，但也有时和时，态和态的构成，那就是：过去将来时、完成进行态。我这里把时态分为一级时态和二级时态

一级时态的分类

一般态

一般态即动作发生时的状态是一般的，平常的，对动作的状态不说其他说明，但对应有不同的时间。其强调时间，不强调动作本身。

一般现在时 v
一般过去时 v-ed
一般将来时 will+v / be going to / be about to（半助动词）

进行态

进行态即表示动作状态是正在进行，更倾向于强调动作发生的过程

现在进行时 be doing
过去进行时 was, were doing
将来进行时 will be doing

完成态

完成态指从某个时间开始一直到现在、过去、将来，并且对现在、过去、将来的结果产生影响的动作，强调动作的结果。

现在完成时 have done
过去完成时 had done
将来完成时 will have done

二级时态的分类

完成进行态

从某个时间开始，一直持续到现在、过去、将来，并且对现在、过去、将来的结果产生影响且还要持续下去的动作，同时强调动作的过程和结果。

现在完成进行时 have done doing
过去完成进行时 had done doing
将来完成进行时 will have done doing

过去将来时

过去将来时与不同的状态构成不同的用法
这种结构的用法通常会用在从句中，“过去”表示主句的谓语发生在过去，“将来”表示从句的谓语发生在“主句谓语过去的将来”，实质上对于说这句话的时间来说还是在过去。

1. 一般过去将来时
一般过去将来时不可以单独使用，通常在宾语从句中，用作间接引语。表示过去的观点看待将来的行为

would/should+v

He asked me if I should be in the office in the afternoon.
他问我下午是否在办公室。

2. 过去将来进行时
从过去某一时间看，未来的某一时刻或某一段时间正在进行的动作

**would/should be+doing

We wrote him that we should be expecting him the following day.
我们写信告诉他第二天我们将等待他。

3. 过去将来完成时
表示过去的将来的某一时间之前完成的动作

would/should+have done

He told me that we should have got to the station by five o' clock.
他告诉我5点前我们就会到达车站。
